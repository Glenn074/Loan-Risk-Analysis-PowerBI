#📊 Loan Risk Analytics & Default Behavior Dashboard
🚀 Business Problem

Financial institutions face significant losses due to loan defaults.
The objective of this project was to:

Identify high-risk borrower segments

Analyze default behavior across states, grades, and loan purposes

Build a risk-focused dashboard to support data-driven lending decisions

🏗️ End-to-End Solution Architecture

This project was built following a complete data analytics pipeline:

Data Cleaning & Transformation – PostgreSQL

Fact & Dimension Table Creation – SQL

Star Schema Data Modeling – Power BI

KPI & Risk Metrics Development – DAX

Interactive Dashboard Development – Power BI

Dataset size: 2.2M+ loan records

🛠️ Tech Stack

PostgreSQL – Data cleaning, transformation, fact table creation

SQL – Data modeling & risk aggregation queries

Power BI – Dashboarding & visualization

DAX – KPI calculations & risk measures

Data Modeling – Star Schema Design

🧱 Data Modeling (Star Schema)

Created a structured analytical model:

FactLoan – Transaction-level loan data

DimLoanDetails – Borrower & loan attributes

Built 1:M relationship using Loan ID

Optimized model for performance & KPI calculation

This improved reporting efficiency and enabled accurate risk segmentation.

📈 Key Risk KPIs Developed

Total Loans Issued

Total Funded Amount

Default Rate %

High Risk Loan Share %

Top Risk Loan Purpose

State-wise Default Distribution

Grade-wise Risk Analysis

📊 Dashboard Structure
1️⃣ Executive Overview

High-level KPIs & portfolio performance summary

2️⃣ Risk Breakdown Analysis

Default behavior by purpose, grade, and verification status

3️⃣ Loan Performance Deep Dive

State-level trends & funded amount vs default correlation

4️⃣ Risk Strategy Insights

Identification of high-risk segments for strategic decision-making

📊 Key Business Insights

Certain loan purposes contribute disproportionately to defaults

Specific states show consistently higher default rates

Lower-grade loans significantly increase portfolio risk

High-risk loan share identified for targeted risk mitigation
